# 🔭 Exoplanet Transit Research

An independent analysis of exoplanet transit parameters using TESS photometry data.
Conducted by a team of 5 students aiming to contribute to exoplanet characterization literature.

## 📋 Research Question
"Independent derivation and verification of transit parameters for [target system] using TESS photometry"

## 📁 Project Structure

exoplanet-transit-research/
├── data/          # Raw TESS light curve data
├── notebooks/     # Jupyter analysis notebooks
├── paper/         # LaTeX paper files
├── figures/       # Generated plots and figures
└── README.md

## 🛠️ Tools & Libraries
- Python 3.x
- Lightkurve — TESS data access
- AstroPy — astronomical calculations
- Matplotlib — plotting
- NumPy — numerical analysis
